# Binary Search Tree #

## Algorithms ##
- insert(node) - O(h)
- remove(node) - O(h)
- findMinNode() - O(h) 
- findMaxNode() - O(h)
- getRootNode() - O(1)
- inOrderTreeWalK(root) / preorder / postorder - O(n)
- search(root, data) - O(h) 

_(h = height of the tree)_
